# pass-the-app
broken phone, but for devs. 
Each day, we would work on a different path with a different project without communicating about the project. 
You can develop whatever you want and add any feature you desire.

## How The Game Would Work
Each project will be developed on a path of branches: 
path[x]/[i]/[dev name]
- '[x]' being the path number
- '[i]' being the step inside the path
- '[dev name]' being the dev that currently works on the project.

In the end, we would go over each project's path and see its progress and what ideas you came up with 🤲🏼
